# Twitter-Feed-Widget
Twitter Feed Widget

```html
<a class="twitter-timeline" href="https://twitter.com/yourtwitterpage" data-widget-id="312402095464652800" target="_blank">Tweets by @yourtwitterpage</a>
<script>
	!function(d,s,id){var js,fjs=d.getElementsByTagName(s)[0];
		if(!d.getElementById(id)){js=d.createElement(s);
			js.id=id;
			js.src="//platform.twitter.com/widgets.js";
			fjs.parentNode.insertBefore(js,fjs);
		}}
		
	(document,"script","twitter-wjs");
</script>
```
